Set it up right (10 minutes)

  1. In a fresh browser window (signed out of personal Google), create the account at accounts.google.com/signup. First name "Apticove", last name "HQ".
    So it is never entangled with a personal login
  2. Generate a long random password in a shared password manager (1Password / Bitwarden) that both of you can open.
    Company asset, not one person's - survives a founder split
  3. Turn on 2-Step Verification with an authenticator app, and store the TOTP seed in the same shared vault so you both can generate codes.
    Solves the "only one phone has the code" problem
  4. Download the backup codes into the shared vault. Set a recovery phone you both control long-term.
    Break-glass access
Why a dedicated account, not a personal one: continuity (the brand's whole login graph never walks out with a personal inbox), a tiny attack surface, and shared control. This account is the single master key - protect it hardest.

Cold-email: do now vs automate later

The cold-email-quickstart system is a 10-phase 0-to-launched wizard, but it needs paid tools you are not on yet. The trick: the cheap physical infrastructure (domains, mailboxes, warmup) is separate from the automation layer. Build the foundation today - start the 4-week warmup clock now - and plug in the paid tools only when you have volume.

What to build now (no paid API, ~$40-90/mo)

  1. Register the cold domains at Cloudflare/Porkbun (~$10-12/yr each): getapticove.com, tryapticove.com, joinapticove.com, apticovehq.com. Never send cold from apticove.com.
    One-time infra, not an API
  2. Provision 8-12 mailboxes (2-3 per cold domain). Maildoso/Zapmail (~$3 each, warmup included) or Google Workspace ($7/seat, highest trust).
    At 15-20 sends/day/mailbox that is 120-240/day once warm
  3. Configure DNS on every cold domain: SPF + DKIM + DMARC (start p=none), a custom tracking subdomain, and a 301 redirect of the bare domain to apticove.com.
    Free, do it the day you buy
  4. Start warmup TODAY - the 4-week clock. Included free in Maildoso/Zapmail; if on raw Workspace, run manual back-and-forth 5-10/day rising to 20-30, always replying.
    Warmup and lead-gen run in parallel - waiting eats a dead month
  5. Write strategy.md by hand: ICP ($500K-$5M US ecommerce brands), offer (the $1,500 teardown wedge), 3 angles (diagnostic hook / peer-proof / specific-mechanism).
    No API
  6. Draft the 4-email sequence + 9 variants by hand: teardown hook → proof → new angle → breakup, ready to paste into the sequencer later.
    No API
The lean manual bridge - first replies with zero sequencer

Hand-send a few highly personalized emails/day from warmed mailboxes, opening with a 5-minute AI Creative Teardown of one real conversion leak on their store. This is your wedge delivered as free value. Personalization at this depth beats any blast.

  • Source 5-10 brands/day (Store Leads free, Apollo free, ad libraries)
  • Verify each against Reoon's free 600/mo before sending
  • Plain text, no links or images in email 1; log in the Sheet; follow up day 3 & 6
  • Reply fast from the same mailbox - two-way conversation is the top trust signal
Buy order, when budget appears
  1. Instantly Growth ($37/mo) - unlocks automated send, warmup, A/B, deploy
  2. Reoon ($11.90 lifetime 10k) - protects the domains you spent a month warming
  3. Consulti ($97/mo) - only once you consume leads faster than you can source by hand
  4. Anthropic + Perplexity keys (pay-as-you-go ~$5-20/mo) - only if manual personalization becomes the bottleneck
  5. Turso / Bison / Store Leads Pro - scale-stage, not now

Minimum viable automated stack: Instantly $37 + Reoon $11.90 one-time = under $40/mo on top of mailboxes.

Bottom line: start the warmup clock and the manual teardown outreach this week. By the time you buy Instantly and load leads, your inboxes are already trusted and your offer is already validated by real replies - so you are scaling something that works, not gambling on a cold start.
